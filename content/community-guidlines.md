FERAL Community Guidelines
Anonymity Rules

Every user gets a randomly assigned name from one master list when they post a story
The assigned name is the same throughout that one thread but changes with every new thread
This means nobody can be tracked or targeted across the community
Any names mentioned INSIDE a story automatically swap to a different name from the same list
Comments display as "Anonymous Member" only — no numbers, no variations, ever
Admin can see hidden session IDs behind the scenes for moderation — users cannot see this

Gender and Identity

Gender field in personal settings is free text, completely optional, never displayed publicly
User types whatever they want or leaves it blank
No dropdowns, no limited options, no required field
You are not required to have a gender to use this app

Moderation

Bans are final
No appeals process
Admin has access to hidden session data to identify repeat bad actors
Terms of Service should make this explicitly clear in both plain and legal language

Community Welcome Message
Before you hit send, make sure you're hitting it with the right words and for the right reasons. Do your best to be understanding. If you have a problem with someone or something, reach out. We're still figuring out how to make this work and we want to get it right. We're in this together.
— Jen Ho 🌿

Community Section — Anonymity System

Every user gets a randomly assigned name from one master list when they post a story
Same name throughout one thread but changes with every new thread so nobody is trackable
Any names mentioned INSIDE a story automatically swap to a different name from the same list
Comments display as "Anonymous Member" only — no numbers, no variations, ever
Admin can see hidden session IDs behind the scenes for moderation — users cannot see this
One master name list — no gender, no choice, randomly assigned every new thread

Gender and Identity

Gender field in personal settings — free text, completely optional, never displayed publicly
No dropdowns, no limited options, no required field
You are not required to have a gender to use this app

Moderation
If something goes wrong in the community we handle it the same way we handle everything else in here — as human beings who are trying. During beta Jen handles this personally. We're learning as we go just like you are. 🌿
Terms of Service — Three Versions
All legal content needs three versions on the same page. User chooses how they take it in.

Full legal text — reviewed by a lawyer before launch
Plain English written version — sounds like Jen, same information
Plain English audio or video version — someone reads the plain English version out loud for people who don't want to read or have trouble reading

Legal jargon lives only in the legal version. Everything else sounds like Jen.
